# 中文打字练习系统

一个基于网页的中文打字练习工具，支持拼音输入练习和句子输入练习，帮助用户提高中文打字速度和准确率。

（首次加载需要联网，因为css和js静态资源首次加载是从网络直接引用的。）

## 新手使用指南（小白必看）

### 第一步：保存练习程序

1. **复制代码**
   - 复制本项目提供的完整 index.html 代码
   
2. **创建文件**
   - 在电脑桌面上点击鼠标右键
   - 选择"新建" → "文本文档"
   - 会出现一个名为"新建文本文档.txt"的文件

3. **粘贴代码**
   - 双击打开这个文本文档
   - 按 `Ctrl+V`（粘贴）将代码粘贴进去
   - 按 `Ctrl+S`（保存）

4. **重命名文件**
   - 关闭文本文档
   - 右键点击这个文件，选择"重命名"
   - 将文件名改为 `index.html`（注意要把 .txt 改成 .html）
   - 如果看不到 .txt 后缀，请参考下面的设置方法

### 第二步：显示文件扩展名（如果需要）

**Windows 10/11 用户：**
1. 打开任意文件夹
2. 点击顶部的"查看"
3. 勾选"文件扩展名"
4. 现在就能看到文件的完整名称了

**Windows 7 用户：**
1. 打开任意文件夹
2. 点击"组织" → "文件夹和搜索选项"
3. 点击"查看"标签
4. 取消勾选"隐藏已知文件类型的扩展名"
5. 点击"确定"

### 第三步：打开练习程序

1. **方法一：直接双击**
   - 找到刚才保存的 `index.html` 文件
   - 双击它，会自动用默认浏览器打开

2. **方法二：右键打开**
   - 右键点击 `index.html` 文件
   - 选择"打开方式"
   - 选择 Chrome、Edge、Firefox 等浏览器

3. **方法三：拖拽到浏览器**
   - 先打开浏览器（Chrome、Edge等）
   - 将 `index.html` 文件拖拽到浏览器窗口中

### 第四步：准备练习资源文件

#### 使用已有词库
1. 可以直接下载项目下的 `chinese-resources.json` 文件，打开页面后选择该文件即可
2. 或者直接复制该文件下的内容，在桌面新建文本文档后，将 `chinese-resources.json` 文件内容直接复制粘贴保存，再重命名该文件为 `xxx.json` json后缀的文件即可使用

#### 创建 JSON 文件（自定义词库创建）

1. **新建文本文档**
   - 在桌面新建一个文本文档
   
2. **输入内容**（复制下面的示例）
   ```json
   {
     "words": ["你好", "谢谢", "学习", "电脑", "练习", "加油", "努力", "进步", "成功", "快乐"],
     "sentences": [
       "今天天气真好。",
       "我喜欢学习新知识。",
       "练习打字很有趣。",
       "坚持就是胜利。",
       "每天都要保持好心情。"
     ]
   }
   ```

3. **保存文件**
   - 点击"文件" → "另存为"
   - 在"文件名"处输入：`练习资源.json`
   - 在"保存类型"处选择"所有文件"
   - 点击"保存"

#### 创建 TXT 文件（更简单）

1. **新建文本文档**
   
2. **输入内容**（每行一个词或句子）
   ```
   你好
   谢谢
   再见
   学习
   电脑
   今天天气很好。
   我正在学习打字。
   练习使人进步。
   ```

3. **保存**
   - 直接按 `Ctrl+S` 保存
   - 文件名可以是 `日常用语.txt`

### 第五步：导入练习资源

1. **打开练习程序**
   - 在浏览器中打开 `index.html`

2. **加载文件**
   - 点击页面上的"加载文件"按钮
   - 在弹出的窗口中找到你创建的 JSON 或 TXT 文件
   - 选中文件，点击"打开"

3. **选择分类**
   - 如果加载成功，下拉菜单中会出现新的选项
   - 选择你刚加载的分类

4. **开始练习**
   - 点击"开始练习"按钮即可

## 功能特点

### 核心功能
- **双阶段练习模式**
  - 第一阶段：拼音输入练习（无声调）
  - 第二阶段：完整句子输入练习
- **实时反馈系统**
  - 键盘按键提示（下一个需要输入的字符会在虚拟键盘上闪烁）
  - 输入正确/错误的即时视觉反馈
  - 完成状态提示
- **灵活的提交机制**
  - 随时按 Enter 键提交当前练习内容
  - 支持未完成提交（会统计为未完成项）
- **详细的统计数据**
  - 实时速度统计（字/分钟）
  - 准确率计算
  - 错误次数统计
  - 未完成提交统计

### 资源管理
- **默认词库**：内置基础练习内容
- **文件导入**：支持导入自定义练习资源
  - 支持 JSON 格式（结构化数据）
  - 支持 TXT 格式（自动识别词汇和句子）
- **多分类管理**：可以在不同的练习分类之间切换

### 用户体验
- **虚拟键盘**：显示标准 QWERTY 键盘布局
- **进度条**：实时显示练习进度
- **成绩评级**：练习结束后根据速度和准确率给出 S/A/B/C 等级
- **响应式设计**：适配不同屏幕尺寸

## 使用方法

1. **开始练习**
   - 点击"开始练习"按钮
   - 确保输入法切换到英文模式

2. **拼音练习阶段**
   - 查看上方显示的中文词汇
   - 输入对应的拼音（无需输入声调）
   - 随时按 Enter 键提交并进入下一个词汇
   - 观察键盘上闪烁的按键提示

3. **句子练习阶段**
   - 完成所有拼音练习后自动进入
   - 输入显示的中文句子
   - 按 Enter 键提交

4. **加载自定义资源**
   - 点击"加载文件"按钮
   - 选择 JSON 或 TXT 文件
   - 在下拉菜单中选择加载的分类

## 资源文件格式

### JSON 格式

支持两种 JSON 格式：

#### 1. 完整格式（带分类）
```json
{
  "categories": {
    "daily": {
      "name": "日常用语",
      "words": ["你好", "谢谢", "再见", "早安", "晚安", "对不起", "没关系", "请问", "帮助", "欢迎"],
      "sentences": [
        "你好，很高兴认识你。",
        "非常感谢你的帮助。",
        "请问现在几点了？",
        "今天天气真不错。",
        "祝你有美好的一天。"
      ]
    },
    "tech": {
      "name": "科技词汇",
      "words": ["电脑", "网络", "软件", "程序", "数据", "系统", "界面", "开发", "测试", "部署"],
      "sentences": [
        "这个软件需要更新到最新版本。",
        "请检查网络连接是否正常。",
        "系统正在进行维护升级。",
        "程序开发需要严格的测试流程。",
        "数据备份是非常重要的工作。"
      ]
    }
  }
}
```

#### 2. 简单格式（单一分类）
```json
{
  "words": ["学习", "练习", "键盘", "打字", "速度", "准确", "提高", "努力", "坚持", "进步"],
  "sentences": [
    "通过不断练习可以提高打字速度。",
    "学习新技能需要持之以恒。",
    "准确率比速度更加重要。",
    "每天坚持练习就会有进步。",
    "熟能生巧是学习的真理。"
  ]
}
```

### TXT 格式

纯文本文件，每行一个词汇或句子：

```
学习
练习
电脑
编程
开发
今天天气很好。
我喜欢学习编程。
练习打字很有趣。
坚持就是胜利。
熟能生巧。
```

**自动识别规则**：
- 包含句号、问号、感叹号等标点符号的识别为句子
- 长度超过 15 个字符的识别为句子
- 包含逗号或顿号的识别为句子
- 其他短文本识别为词汇
- 文件名将作为分类名称显示

## 技术实现

- **前端技术**：纯 HTML5 + CSS3 + JavaScript
- **拼音库**：[pinyin-pro](https://github.com/zh-lx/pinyin-pro) - 用于生成汉字拼音
- **无需后端**：所有功能均在浏览器端实现
- **数据存储**：练习数据保存在浏览器内存中

## 评分标准

练习结束后会根据表现给出评级：

- **S 级**：准确率 ≥ 95% 且 速度 ≥ 60 字/分钟
- **A 级**：准确率 ≥ 90% 且 速度 ≥ 40 字/分钟
- **B 级**：准确率 ≥ 80% 且 速度 ≥ 25 字/分钟
- **C 级**：其他情况

## 统计说明

- **速度**：每分钟正确输入的字符数
- **准确率**：正确输入的字符数 ÷ 总输入字符数 × 100%
- **错误次数**：总字符数 - 正确字符数
- **未完成提交**：未输入完整就按 Enter 提交的次数

## 常见问题

### Q: 为什么打开后是乱码？
A: 请确保文件保存为 UTF-8 编码格式。在记事本中，保存时选择"UTF-8"编码。

### Q: 为什么双击打开显示的是代码？
A: 文件扩展名没有改成 .html。请确保文件名是 `index.html` 而不是 `index.html.txt`。

### Q: 拼音练习时为什么输入不了？
A: 请确保输入法切换到英文模式。通常按 Shift 键可以切换。

### Q: 加载文件失败怎么办？
A: 
1. 检查文件格式是否正确（JSON 文件要有正确的格式）
2. 确保文件扩展名是 .json 或 .txt
3. JSON 文件中的引号必须是英文引号 `"` 而不是中文引号 `"`

### Q: 可以用手机练习吗？
A: 本系统主要为电脑键盘设计，手机使用体验可能不佳。

## 注意事项

1. 练习拼音时请确保输入法处于英文模式
2. 练习句子时请切换到中文输入法
3. 支持的文件格式：.json 和 .txt
4. 建议每次练习 10-20 个词汇，避免疲劳
5. 定期练习才能有效提高打字速度
6. 练习资源文件请保存在容易找到的位置（如桌面）
